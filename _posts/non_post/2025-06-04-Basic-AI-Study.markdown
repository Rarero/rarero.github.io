## AI 기초 스터디 개요

이 포스트는 AI(인공지능)를 처음 접하는 사람들을 위한 기초 스터디 기록입니다.  
기초 개념부터 시작하여, 머신러닝과 딥러닝의 차이, 주요 알고리즘, 실습 환경 구성까지 실용적인 학습 내용을 다룹니다.

## 1. AI란 무엇인가?

AI(Artificial Intelligence)는 인간의 지능을 모방하여 문제를 해결하거나 학습을 수행하는 시스템을 의미합니다.  
대표적인 예로는 챗봇, 이미지 인식, 추천 시스템 등이 있으며, 세부 기술로는 머신러닝(ML), 딥러닝(DL)이 있습니다.

## 2. 머신러닝 vs 딥러닝

| 구분 | 머신러닝 | 딥러닝 |
|------|----------|---------|
| 정의 | 데이터를 기반으로 패턴을 학습하는 알고리즘 | 인공신경망 기반의 심층 학습 방식 |
| 주요 알고리즘 | 선형 회귀, 의사결정나무, SVM 등 | CNN, RNN, Transformer 등 |
| 데이터 요구량 | 상대적으로 적음 | 대용량 데이터 필요 |
| 계산 자원 | 적음 | GPU 등 고성능 자원 필요 |

## 3. 대표 알고리즘

- **지도학습 (Supervised Learning)**: 라벨이 있는 데이터를 기반으로 학습  
  예: 이미지 분류, 이메일 스팸 필터링

- **비지도학습 (Unsupervised Learning)**: 라벨 없는 데이터를 기반으로 군집화, 패턴 발견  
  예: 고객 세그먼트 분석, 이상 탐지

- **강화학습 (Reinforcement Learning)**: 보상을 기반으로 최적의 행동을 학습  
  예: 게임 에이전트, 로봇 제어

## 4. 실습 환경 구성

- **Python + Jupyter Notebook**: AI 개발 표준 환경
- **필수 라이브러리**:
  - `numpy`, `pandas` (데이터 처리)
  - `scikit-learn` (기본 ML 알고리즘)
  - `matplotlib`, `seaborn` (시각화)
- **클라우드 환경** (선택):
  - Google Colab
  - Azure Notebooks
  - VSCode + Azure ML
